# Лабораторная работа №4  
**Классификация и кластеризация данных**

В репозитории представлено решение лабораторной работы №4 по машинному обучению на датасете музыкальных треков Spotify.

## Содержание работы
- **Обучение с учителем**: классификация признака `explicit`
- **Обучение без учителя**: кластеризация без использования целевой метки

## Используемые методы

### Классификация
- kNN (подбор числа соседей)
- Logistic Regression
- Random Forest  
Учитывалась несбалансированность классов и корреляция признаков.

### Кластеризация
- KMeans  
- DBSCAN  
- Agglomerative clustering (Ward)

Для визуализации использовалось понижение размерности **PCA**.  
Результаты кластеризации сравнивались с истинными метками с помощью **ARI** и **NMI**.

## Файлы
- `lab4_fixed_and_clustering.ipynb` — основной ноутбук с решением  
- `spotify_data_clean.csv` — датасет  
- `12-6. Оценка стоимости квартиры.ipynb` — пример из методических материалов

## Вывод
Признак `explicit` не образует выраженной кластерной структуры в пространстве аудиофичей, что подтверждается низкими значениями ARI и NMI для всех алгоритмов кластеризации.
